ADT7410 Temperature Reader for Linux
====================================

How to use
----------

```
git clone https://github.com/informationsea/ADT7410.git
cd ATD7410
make
sudo chown root:root ADT7410
sudo chmod gu+s ADT7410
./ADT7410
```

Tested Devices & Software
--------------

* Raspberry Pi Model B+
* Arch Linux
